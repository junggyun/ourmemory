## 👉 프로젝트 소개
다양한 주제의 그룹을 생성할 수 있으며 그룹원의 참여 코드 공유를 통해 그룹에 참여하여 <br>
게시글, 댓글기능을 이용할 수 있는 프라이빗 커뮤니티입니다.
